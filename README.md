# 单词王比赛 - 小学生英语学习平台 ⚡

> 一个专为小学生设计的英语单词学习平台，采用可爱的宝可梦主题界面 🎮

[![Version](https://img.shields.io/badge/version-1.6.0-blue.svg)](https://github.com)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

## ✨ 主要功能

### 🎯 三种学习模式
- **📚 背单词模式** - 系统化学习新单词，加深记忆
- **✍️ 检查模式** - 测试单词拼写和记忆
- **⚡ 挑战模式** - 限时单词挑战赛，提升速度

### 🌟 特色功能
- **⚔️ 斩词系统** - 标记已掌握的单词，跟踪学习进度
- **📕 错词本** - 自动收集错误单词，针对性复习
- **📅 学习日历** - 可视化学习记录，培养学习习惯
- **📥 Excel 导入** - 支持自定义单词库导入（.xlsx, .csv）
- **🔢 灵活选择** - 自定义学习单词数量

### 🎨 界面特点
- **宝可梦主题** - 明亮色彩，童趣十足
- **22+ 动画效果** - 生动有趣的交互体验
- **响应式设计** - 支持各种屏幕尺寸
- **本地存储** - 数据安全保存在浏览器

## 🚀 快速开始

### 在线体验
直接访问部署后的网址即可开始使用，无需安装。

### 本地运行
1. 克隆或下载项目
2. 在项目目录下启动本地服务器：
   ```bash
   python3 -m http.server 8000
   # 或
   ./start.sh
   ```
3. 浏览器访问 `http://localhost:8000`

## 📖 使用说明

### 基础使用
1. **选择模式** - 在主菜单选择学习模式
2. **选择话题** - 选择单词话题分类，或输入数量随机选词
3. **开始学习** - 根据提示进行学习或测试
4. **查看成绩** - 学习结束后查看统计数据

### Excel 导入格式
支持的列名（中英文均可）：
- **单词/Word** - 英文单词（必填）
- **中文/Translation** - 中文释义（必填）
- **话题/Category** - 单词分类（可选）
- **音标/Phonetic** - 音标（可选）
- **例句/Example** - 例句（可选）

## 🎯 技术栈

- **前端框架**: 原生 HTML5 + CSS3 + JavaScript
- **样式设计**: 自定义宝可梦主题 CSS
- **数据存储**: localStorage
- **Excel 解析**: SheetJS (xlsx.js)
- **部署平台**: Vercel
