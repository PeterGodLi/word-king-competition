# 📊 单词数量选择功能使用说明

## 🎯 功能概述

在所有学习模式中，新增了**单词数量选择**功能，允许您指定学习的单词数量，系统将随机从所选话题中挑选指定数量的单词进行学习。

---

## ✨ 适用场景

本功能适用于以下所有学习模式：

1. **📚 背单词模式**
2. **✍️ 检查模式**  
3. **⚡ 挑战模式**
4. **📕 背诵错词**
5. **📕 检查错词**

---

## 📖 使用方法

### 一、在话题选择界面使用

#### 1️⃣ 启动学习模式

从主菜单选择任意学习模式（背单词、检查、挑战）

#### 2️⃣ 选择话题

在话题选择界面，点击您想学习的话题分类

#### 3️⃣ 设置数量（可选）

在顶部的 **🔢 单词数量** 输入框中输入您想学习的单词数量：
- **输入数字**：例如输入 `10`，则随机选择 10 个单词
- **留空**：不输入任何内容，则选择全部单词

#### 4️⃣ 开始学习

点击"开始学习"按钮

---

### 二、在错词本中使用

#### 1️⃣ 打开错词本

点击顶部的 **📕 [数字]** 按钮打开错词本管理器

#### 2️⃣ 设置练习数量（可选）

在 **🔢 练习数量** 输入框中输入您想练习的错词数量：
- **输入数字**：例如输入 `5`，则随机选择 5 个错词
- **留空**：不输入任何内容，则练习全部错词

#### 3️⃣ 选择模式

点击"📚 背诵错词"或"✍️ 检查错词"按钮开始练习

---

## 🌟 功能特点

### 1. **智能随机选择**
系统会从所有符合条件的单词中**随机挑选**指定数量的单词，每次学习都有不同的组合。

### 2. **灵活性强**
- 可以指定精确数量（如 10、20、50 等）
- 也可以不限制数量（留空）
- 适合不同的学习时长需求

### 3. **自动调整**
- 如果输入的数量**大于可用单词总数**，系统会自动使用全部单词
- 如果输入的数量**小于等于 0**，系统会提示输入有效数字

### 4. **跨模式支持**
所有学习模式都支持数量限制，使用体验一致

## 🔢 输入规则

### ✅ 有效输入

- **正整数**：`1`、`10`、`50`、`100` 等
- **留空**：不输入任何内容（选择全部）

### ❌ 无效输入

- **0 或负数**：`0`、`-5` 等 → 系统提示错误
- **小数**：`10.5` → 系统自动取整为 10
- **非数字**：`abc` → 系统忽略

## ⚙️ 技术细节

### 随机算法

使用 **Fisher-Yates 洗牌算法**确保：
- 每个单词被选中的概率相等
- 每次学习的单词组合都不同
- 真正的随机分布

### 数量处理逻辑

```javascript
// 伪代码示例
if (输入数量 > 0 && 输入数量 < 总单词数) {
    随机选择指定数量的单词
} else if (输入数量 >= 总单词数) {
    使用全部单词
} else {
    使用全部单词（默认行为）
}
```

## 🛠️ 常见问题

### Q1: 输入的数量大于可用单词怎么办？

**A:** 系统会自动使用全部可用单词。例如，某话题只有 15 个单词，您输入 50，系统会使用这 15 个单词。

---

### Q2: 能否保证每次学习的单词都不一样？

**A:** 系统使用真随机算法，每次学习都会重新随机选择，所以几乎不会出现完全相同的组合（除非单词总数很少）。

---

### Q3: 数量限制会影响学习效果吗？

**A:** 不会。数量限制只是帮助您控制学习时长，系统会随机选择单词，确保学习的全面性。

---

### Q4: 错词本数量很少时怎么办？

**A:** 可以输入小数量（如 3-5 个）进行快速复习，或者留空学习全部错词。

---

### Q5: 可以输入小数吗？

**A:** 可以输入，但系统会自动取整。例如输入 `10.7` 会被处理为 `10`。
