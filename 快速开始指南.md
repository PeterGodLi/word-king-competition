## 2. 使用学习模式

### 📚 背单词模式

1. 在主菜单点击 **"📚 背单词模式"**
2. 选择一个或多个话题（点击卡片选中）
3. 点击 **"开始学习"**
4. 使用功能：
   - 🔊 点击喇叭按钮听发音
   - 😊 点击"认识"记录已掌握
   - 😕 点击"不认识"标记需复习
   - ➡️ 点击"下一个"继续学习

**适合场景**：初次学习新单词，建立记忆

### ✍️ 检查模式

1. 在主菜单点击 **"✍️ 检查模式"**
2. 选择要测试的话题
3. 点击 **"开始学习"**
4. 使用方法：
   - 根据中文释义输入英文单词
   - 🔊 可以听发音提示
   - ✅/❌ 即时查看答案对错
   - Enter键快速提交

**适合场景**：测试单词记忆，检验学习效果

### ⚡ 挑战模式

1. 在主菜单点击 **"⚡ 挑战模式"**
2. 选择挑战的话题
3. 点击 **"开始学习"**
4. 挑战规则：
   - ⏱️ 60秒倒计时
   - 根据中文快速输入单词
   - 连续答对获得连击加分
   - Enter键快速提交答案

**适合场景**：提升反应速度，增加学习趣味性

## 3. 导入自定义单词库

### 准备Excel文件

创建Excel文件，包含以下列：

| 列名（中文） | 列名（英文） | 必填 | 说明 |
|------------|------------|------|------|
| 单词 | Word | ✅ | 英文单词 |
| 中文 | Translation | ✅ | 中文释义 |
| 话题 | Category | ⭕ | 分类名称 |
| 音标 | Phonetic | ⭕ | 发音音标 |
| 例句 | Example | ⭕ | 使用例句 |

**示例数据**：

```
话题      单词     音标        中文      例句
交通工具  car      /kɑː/       汽车      I go to school by car.
交通工具  bus      /bʌs/       公共汽车   The bus is coming.
天气      sunny    /ˈsʌni/     晴朗的     It's sunny today.
```

### 导入步骤

1. **打开导入界面**
   - 点击主菜单的 **"📥 导入单词库"**

2. **上传文件**
   - 方法1：点击上传区域选择文件
   - 方法2：拖拽文件到上传区域
   - 支持格式：.xlsx, .xls, .csv

3. **查看解析进度**
   - 系统自动解析文件
   - 显示实时进度条
   - 自动识别列名

4. **预览数据**
   - 查看统计信息（分类数、单词数）
   - 预览每个分类的单词
   - 检查是否有错误提示

5. **确认导入**
   - 确认数据无误
   - 点击 **"导入单词库"**
   - 等待导入完成提示

6. **使用导入的单词**
   - 返回主菜单
   - 在任意学习模式中选择导入的话题
   - 开始学习！

### 快速测试

项目中包含 `示例单词库.csv`，可直接导入测试：


## 4. 常见问题

### Q1: 无法听到单词发音？

**A:** 
- 检查浏览器是否支持Web Speech API
- 推荐使用Chrome、Edge或Safari
- 确保设备音量已打开

### Q2: 导入的单词在哪里？

**A:**
- 返回主菜单
- 选择任意学习模式
- 在话题选择界面可以看到导入的分类

### Q3: 导入失败怎么办？

**A:**
- 检查Excel格式是否正确
- 确保包含"单词"和"中文"列
- 查看错误提示信息
- 尝试使用CSV格式

### Q4: 如何删除导入的单词？

**A:**
- 方法1：清除浏览器缓存
- 方法2：在浏览器控制台执行：
  ```javascript
  localStorage.removeItem('customWordDatabase')
  ```
- 然后刷新页面

### Q5: 导入的数据会丢失吗？

**A:**
- 数据保存在浏览器本地存储中
- 关闭浏览器后仍然保留
- 建议保留原始Excel文件备份
- 清除浏览器数据会删除导入的单词

### Q6: 可以同时导入多个文件吗？

**A:**
- 一次只能导入一个文件
- 但可以多次导入不同文件
- 数据会自动合并（相同分类会合并，重复单词会去重）

### Q7: 支持哪些Excel版本？

**A:**
- ✅ Excel 2007及更高版本 (.xlsx)
- ✅ Excel 97-2003 (.xls)
- ✅ CSV格式 (.csv)

### Q8: 音标格式有要求吗？

**A:**
- 可以带斜杠：/kɑː/
- 也可以不带：kɑː
- 程序会自动添加斜杠格式

### Q9: 列名必须完全匹配吗？

**A:**
- 不需要！程序支持智能识别
- 支持中文：单词、中文、话题、音标、例句
- 支持英文：Word, Translation, Category, Phonetic, Example
- 大小写不敏感

### Q10: 如何创建CSV文件？

**A:**
- Excel中"另存为" → 选择CSV格式
- 使用记事本/文本编辑器创建
- 第一行为列名，后续行为数据
- 建议使用UTF-8编码

## 💡 使用技巧

### 1. 高效学习策略

```
第一步：背单词模式（建立印象）
   ↓
第二步：检查模式（验证记忆）
   ↓
第三步：挑战模式（巩固提升）
```

### 2. 分类管理

- 按主题分类：动物、食物、颜色等
- 按难度分类：入门、进阶、高级
- 按教材分类：第一单元、第二单元等

### 3. 自定义例句

- 使用学生熟悉的场景
- 结合生活实际
- 中英文对照便于理解

### 4. 键盘快捷键

- **Enter** - 提交答案（检查模式、挑战模式）
- **ESC** - 关闭导入对话框
- **Tab** - 切换输入框

### 5. 进度跟踪

- 查看得分统计
- 记录认识/不认识的单词
- 查看正确率和完成度
