## 🎯 文件使用场景

### 场景1: 首次使用

```
必读文件:
  ✅ README.md
  ✅ 快速开始指南.md

操作步骤:
  1. 执行 ./start.sh 启动应用
  2. 浏览器访问 http://localhost:8000
  3. 尝试三种学习模式
  4. 导入 示例单词库.csv 测试
```

### 场景2: 导入自定义单词

```
参考文件:
  ✅ Excel导入使用说明.md
  ✅ 示例单词库.csv (作为模板)

操作步骤:
  1. 参照示例创建Excel文件
  2. 点击"📥 导入单词库"
  3. 上传文件并预览
  4. 确认导入
```

### 场景3: 开发测试

```
使用文件:
  ✅ test-import.html
  ✅ 测试_含错误数据.xlsx
  ✅ 测试_大量数据.xlsx

测试流程:
  1. 打开test-import.html
  2. 查看功能测试结果
  3. 上传测试文件验证

```
使用脚本:
