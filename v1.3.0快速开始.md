# 🚀 v1.3.0 快速开始指南

## 📌 新功能快速体验

### 🎯 功能一：话题学习 + 数量限制

**30秒快速上手**

1. **启动应用**
   ```bash
   打开浏览器访问: http://localhost:8000
   ```

2. **选择模式**
   - 点击 `📚 背单词`、`✍️ 检查` 或 `⚡ 挑战`

3. **设置数量**（新功能！）
   - 在 `🔢 单词数量` 输入框输入：`10`
   - 或留空（选择全部单词）

4. **选择话题**
   - 点击任意话题，如 `🍎 食物与饮料`

5. **开始学习**
   - 点击 `开始学习` 按钮
   - **系统会从所选话题中随机抽取 10 个单词！**

---

### 🎯 功能二：错词本 + 练习数量

**30秒快速上手**

1. **打开错词本**
   - 点击顶部 `📕 [数字]` 按钮

2. **设置数量**（新功能！）
   - 在 `🔢 练习数量` 输入框输入：`5`
   - 或留空（练习全部错词）

3. **选择模式**
   - 点击 `📚 背诵错词` 或 `✍️ 检查错词`

4. **开始练习**
   - **系统会从错词本中随机抽取 5 个错词！**

## ⚡ 快速操作指南

### 方式 1：精准学习（推荐）
1. 选择 1-2 个话题
2. 输入具体数量（如 `20`）
3. 专注学习固定数量

### 方式 2：随心学习
1. 选择多个话题
2. **留空不输入数量**
3. 学习所有单词

### 方式 3：混合学习
1. 选择 3-4 个话题
2. 输入中等数量（如 `30`）
3. 多样化学习内容

## 🔢 输入提示

### ✅ 有效输入
- `10` → 精确学习 10 个单词
- `50` → 学习 50 个单词
- 留空 → 学习全部单词

### ❌ 无效输入
- `0` → 提示错误
- `-5` → 提示错误
- `abc` → 自动视为留空
