## 🎉 新功能概述

### 功能一：优化斩词功能
- **一键斩词**：无需确认，点击即可斩词并自动跳转到下一个单词
- **实时反馈**：斩词后显示简洁的Toast提示
- **无缝学习**：斩词不中断学习流程，自动继续下一个单词

### 功能二：错词本功能
- **自动记录**：背单词和检查模式中答错的单词自动添加到错词本

- **数据导出**：支持导出错词本为CSV文件

## 📚 功能详细说明

### 一、优化后的斩词功能

#### 使用方法

1. **在背单词模式中**
   - 学习单词时，点击页面底部的 `⚔️ 斩词` 按钮
   - 单词立即被标记为"已掌握"并从学习列表中移除
   - 自动跳转到下一个单词，无需额外操作

2. **斩词效果**
   - 单词被添加到斩词列表
   - 获得 +10 经验值
   - 弹出Toast提示："单词 'xxx' 已斩！"
   - 自动显示下一个单词

3. **特点优势**
   - ✅ 无需二次确认，操作更快捷
   - ✅ 不打断学习节奏
   - ✅ 第一次学习就可以斩词
   - ✅ 已斩单词不会在后续学习中出现

#### 注意事项
- 斩词操作不可撤销（只能在斩词管理中取回）
- 建议确实掌握的单词再斩词
- 斩词数据保存在浏览器本地存储中

### 二、错词本功能

#### 错词自动收集

**触发场景：**

1. **背单词模式**
   - 点击"😕 不认识"按钮时
   - 单词会被自动添加到错词本

2. **检查模式**
   - 输入错误答案时
   - 系统自动记录到错词本

3. **错误统计**
   - 重复错误会累加错误次数
   - 记录最后一次错误时间
   - 按错误次数排序显示

#### 错词本管理

**打开错词本**
- 点击顶部导航栏的 `📕 [数字]` 按钮
- 数字显示当前错词数量

**界面功能**

```
┌─────────────────────────────────┐
│  📕 错词本                       │
├─────────────────────────────────┤
│  统计卡片：                      │
│  - 错误单词数                    │
│  - 累计错误次数                  │
├─────────────────────────────────┤
│  快速操作：                      │
│  [📚 背诵错词]  [✍️ 检查错词]   │
├─────────────────────────────────┤
│  错词列表：                      │
│  - 显示单词、音标、释义           │
│  - 显示错误次数标识               │
│  - 显示分类和时间                 │
│  - 可搜索、移除单词               │
└─────────────────────────────────┘
```

**错词列表说明**

每个错词条目显示：
- 📝 单词和音标
- 🔢 错误次数徽章（如："3次"）
- 📖 中文释义
- 📁 所属分类
- ⏰ 添加时间（相对时间）
- ✕ 移除按钮

#### 错词专项练习

**1. 背诵错词**
- 点击 `📚 背诵错词` 按钮
- 以背单词模式学习错词本单词
- 可以标记"认识"或"不认识"
- 可以继续斩词（如果真的掌握了）

**2. 检查错词**
- 点击 `✍️ 检查错词` 按钮
- 以检查模式测试错词本单词
- 看中文拼写英文
- 答错会继续累加错误次数

**3. 学习特点**
- 🎯 针对性强，专攻薄弱环节
- 📊 按错误次数排序，优先学习难点
- ♻️ 可重复练习，直到掌握
- 📈 实时统计，了解进步情况

#### 数据管理

**搜索功能**
- 在搜索框输入单词
- 实时过滤显示匹配的错词
- 支持模糊匹配

**移除错词**
- 点击单词旁的 `✕ 移除` 按钮
- 确认后从错词本删除
- 适用于已经掌握的单词

**清空错词本**
- 点击 `清空全部` 按钮
- 需要两次确认（谨慎操作）
- 清空后不可恢复

**导出错词**
- 点击 `📥 导出数据` 按钮
- 下载CSV格式文件
- 包含完整信息：
  - 单词、音标、中文、分类、例句
  - 错误次数、添加时间、最后错误时间
- 文件名格式：`错词本_2025-11-26.csv`

## 💡 使用建议

### 斩词策略
1. **及时斩词**：确实掌握的单词立即斩掉，提高学习效率
2. **定期复习**：在斩词管理中查看已斩单词，确保没有遗忘
3. **合理判断**：不要为了快速完成而随意斩词

### 错词本策略
1. **重点攻克**：优先练习错误次数多的单词
2. **循环学习**：定期进行错词本专项练习
3. **及时清理**：掌握的单词及时移除，保持错词本质量
4. **数据导出**：定期导出错词本数据作为学习记录

### 组合使用
```
学习流程建议：
1. 选择分类进行常规学习
2. 遇到简单单词 → 直接斩词
3. 遇到难单词 → 标记不认识（自动加入错词本）
4. 定期打开错词本进行专项练习
5. 错词本单词掌握后 → 斩词或移除
6. 导出错词本数据留存学习记录
```

## ❓ 常见问题

**Q1：斩词后能恢复吗？**
A：可以。在斩词管理界面中找到对应单词，点击"取回"按钮即可恢复到学习列表。

**Q2：错词会重复添加吗？**
A：同一个单词多次答错时，不会重复添加，只会累加错误次数。

**Q3：清空错词本后能恢复吗？**
A：不能。建议清空前先导出数据备份。

**Q4：导出的CSV文件如何使用？**
A：可以用Excel、Numbers等软件打开，也可以重新导入到单词王系统。

**Q5：错词本和斩词有什么区别？**
A：
- 斩词：表示已经掌握的单词，不再出现
- 错词本：表示需要加强的单词，可反复练习

**Q6：可以同时打开斩词管理和错词本吗？**
A：不可以，它们是独立的弹窗界面，同一时间只能打开一个。

**Q7：数据会丢失吗？**
A：数据保存在浏览器本地，除非清除浏览器数据，否则不会丢失。建议定期导出备份。
