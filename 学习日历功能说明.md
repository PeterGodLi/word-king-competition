# 📅 学习日历功能说明

## 功能概述

学习日历功能以可视化日历的形式记录每天的学习数据，帮助孩子和家长直观地了解学习进度和效果。

---

## ✨ 主要特色

### 1. 自动记录学习数据
系统会自动记录以下信息：
- 📚 **背单词数量** - 学习模式完成的单词数
- ✍️ **测试单词数量** - 检查模式完成的单词数
- ⚡ **挑战模式数量** - 挑战模式完成的单词数
- ⭐ **总得分** - 累计获得的分数
- ✅ **正确数量** - 答对的题目数
- ❌ **错误数量** - 答错的题目数
- ⚔️ **斩词数量** - 标记为已掌握的单词数
- ⏱️ **学习时长** - 当天学习的总时间（分钟）

### 2. 可视化日历展示
- 📅 按月份展示学习记录
- 🌈 不同颜色表示不同的学习强度：
  - **浅绿色** - 1-19个单词
  - **橙黄色** - 20-49个单词
  - **紫罗兰** - 50个以上单词
- ✨ 有学习记录的日期显示星星图标
- 🎯 今天的日期特别高亮显示

### 3. 详细统计信息
点击任意日期可查看当天详细数据：
- 各模式完成的单词数
- 得分情况
- 正确率统计
- 学习时长

### 4. 数据导出
- 📥 支持导出为CSV文件
- 📊 方便用Excel等工具进一步分析
- 📈 家长可以跟踪孩子的学习曲线

---

## 🎮 使用方法

### 访问日历
1. 点击页面顶部的 **📅 日历** 按钮
2. 打开学习日历界面

### 查看统计
日历顶部显示总体统计：
- 📅 学习天数 - 总共学习了多少天
- 📚 累计单词 - 学习+测试+挑战的总单词数
- ⭐ 总得分 - 累计获得的分数
- ⏱️ 学习分钟 - 累计学习时长

### 查看日期详情
1. 点击日历中的任意日期
2. 弹出详情面板，显示当天的学习数据
3. 包括：背单词、测试、挑战、得分、正确、错误、斩词、时长、正确率

### 导出数据
1. 点击日历底部的 **📥 导出数据** 按钮
2. 自动下载CSV文件
3. 文件名格式：`学习日历_2025-11-26.csv`

## 📊 数据记录逻辑

### 自动记录时机
1. **完成学习** - 背单词模式结束时
2. **完成测试** - 检查模式结束时
3. **完成挑战** - 挑战模式结束时
4. **斩词操作** - 每次斩词时

### 数据累加规则
- 同一天内多次学习，数据会累加
- 例如：上午学10个，下午学20个 = 当天30个
- 分数、正确数、错误数都会累计

### 数据存储
- 使用 `localStorage` 本地存储
- 数据格式：`{ '2025-11-26': { learnCount, testCount, ... } }`
- 数据永久保存，除非手动清空

### 数据结构
```javascript
{
  '2025-11-26': {
    date: '2025-11-26',
    learnCount: 20,        // 背单词数量
```

## ❓ 常见问题

### Q1: 数据会丢失吗？
A: 数据存储在浏览器本地，除非清除浏览器数据或手动清空，否则永久保存。

### Q2: 可以修改历史数据吗？
A: 目前不支持手动修改，所有数据都是系统自动记录的。

### Q3: 导出的CSV文件如何使用？
A: 可以用Excel、Numbers等软件打开，进行数据分析和图表制作。

### Q4: 为什么今天的数据不准确？
A: 确保完成学习后看到结果页面，数据才会被记录。中途退出不会记录。

### Q5: 可以查看以前月份的数据吗？
A: 当前版本只显示本月，可以通过导出CSV查看所有历史数据。
