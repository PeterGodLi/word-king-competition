📋 功能概述
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

斩词功能允许用户标记已完全掌握的单词，被标记的单词将：
  ✅ 从所有学习模式中自动排除
  ✅ 保存到斩词管理器中
  ✅ 支持随时取回恢复
  ✅ 导出学习记录

🎯 演示流程
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【步骤1】启动应用并进入学习模式
  1. 启动应用: ./start.sh
  2. 在浏览器访问: http://localhost:8000
  3. 点击"📚 背单词模式"
  4. 选择任意话题（如：食物与饮料）
  5. 点击"开始学习"

【步骤2】展示斩词操作
  1. 学习几个单词
  2. 遇到已掌握的单词时，点击"⚔️ 斩词"按钮
  3. 确认斩词操作
  4. 观察效果：
     ✨ 炫酷的剑斩动画
     🎯 显示"单词已斩！+10经验值"
     ➡️ 自动跳到下一个单词
     📊 右上角斩词计数器 +1

【步骤3】演示斩词管理器
  1. 点击页面右上角的"⚔️ [数字]"按钮
  2. 打开斩词管理界面
  3. 展示统计信息：
     - 已斩单词数量
     - 单词总数
     - 完成率百分比
  4. 展示已斩单词列表：
     - 单词详细信息
     - 斩词时间
     - 所属分类

【步骤4】演示搜索功能
  1. 在搜索框输入单词
  2. 实时过滤显示结果
  3. 清空搜索框显示全部

【步骤5】演示取回功能
  1. 找到一个已斩单词
  2. 点击"↩️ 取回"按钮
  3. 确认取回操作
  4. 观察效果：
     ✅ 显示成功提示
     📉 统计数据自动更新
     📚 单词恢复到学习中

【步骤6】演示导出功能
  1. 点击"📥 导出数据"按钮
  2. 自动下载CSV文件
  3. 用Excel打开查看（可选）

【步骤7】验证过滤效果
  1. 关闭斩词管理器
  2. 返回主菜单
  3. 再次进入相同话题的学习
  4. 确认已斩单词不再出现
