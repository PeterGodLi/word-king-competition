## ⚡ 10秒快速体验

### 方式一：快速模式（新功能）

1. **打开应用**
   ```
   http://localhost:8000
   ```

2. **选择模式**
   - 点击 `📚 背单词`

3. **输入数量**
   - 在顶部输入框输入：`10`

4. **开始学习**
   - 直接点击 `开始学习`
   - **无需选择话题！** ✨

5. **体验效果**
   - 系统从全量词库随机选择 10 个单词
   - 单词来自不同话题（动物、颜色、数字等）
   - 每次学习内容都不同

---

### 方式二：话题模式（传统方式）

1. **选择模式**
   - 点击 `✍️ 检查`

2. **不输入数量**
   - 留空输入框

3. **选择话题**
   - 点击 `🍎 食物与饮料`

4. **开始学习**
   - 点击 `开始学习`

5. **体验效果**
   - 学习"食物与饮料"话题的全部 6 个单词

## ⚠️ 注意事项

### 1. 两种模式的优先级

**规则**：
```
如果输入了数量（> 0）
  → 自动从全库选择，话题选择失效

如果没有输入数量
  → 必须选择话题，否则提示错误
```

---

### 2. 数量输入建议

**推荐范围**：
```
碎片时间: 5-10 个
短时学习: 10-20 个
中等强度: 20-40 个
深度学习: 40-80 个
```

**不推荐**：
```
❌ 输入太少（< 5）：效果不明显
❌ 输入太多（> 100）：容易疲劳
```

---

### 3. 错词本使用

**仍需输入数量**：
```
错词本 → 输入数量 → 选择模式
（不会自动从全库选择）
```

## 🔧 常见问题

### Q1: 输入数量后还能选择话题吗？

**A:** 可以选择，但不会生效。一旦输入数量，系统会优先从全量词库中随机选择单词。

---

### Q2: 如何只学习特定话题？

**A:** 不要输入数量（留空），然后选择您想学习的话题即可。

---

### Q3: 输入数量后，每次学习的单词会重复吗？

**A:** 不会。每次都会重新随机选择，单词组合和顺序都会不同。

---

### Q4: 全量词库有多少个单词？

**A:** 当前内置词库有 8 个话题，共 48 个单词。如果导入了自定义词库，数量会更多。

---

### Q5: 可以同时使用数量限制和话题选择吗？

**A:** 在 v1.4.0 中，输入数量会优先生效，话题选择会被忽略。如果需要限制话题的数量，请使用话题模式（不输入数量）。
